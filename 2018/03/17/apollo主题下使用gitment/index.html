apollo主题下使用gitment

2018年3月17日

前言

之前一直没在blog下添加评论系统，多说貌似不干了，偶然间发现了gitment，它是一款基于GitHub Issues 的评论系统，同时有完整的 Markdown / GFM 和代码高亮支持。感觉不错，就添加到blog下了。

方法

1. 注册 OAuth Application

点击此处 来注册一个新的 OAuth Application。其他内容可以随意填写，但要确保填入正确的 callback URL(一般是评论页面对应的域名，如 https://yangsoon.github.io)。你会得到一个 client ID 和一个 client secret，这个将被用于之后的用户登录。

2. 引入Gitment

因为使用的是apollo主题，所以我们先配置 _config.yml 文件。只需要添加几行代码就行。
